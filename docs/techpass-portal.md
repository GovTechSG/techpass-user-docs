# TechPass portal

You can use [TechPass portal](https://portal.techpass.gov.sg) to:

- Sign up for a TechPass account if you have a non-SE GSIB device.
- SGTS tenant admins can set up SSO for their SGTS products and manage IAM for their users.

Refer to [TechPass Tenant Guide](https://docs.developer.tech.gov.sg/docs/techpass-tenant-guide/#/) to learn more about setting up SSO and manage users.

?> You need to log in with your TechPass account to view this user guide.
